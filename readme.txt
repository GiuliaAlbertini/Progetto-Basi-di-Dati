Prima dell'esecuzione:
- aprire in MySQL Workbench lo script SQL "ELABORATO_BASI.sql" ed eseguirlo.
Al suo interno contiene le istruzioni necessarie alla creazione delle tabelle
e alla definizione dei vincoli referenziali, così come le query per popolare
il database.

Per l'esecuzione:
- lanciare l'applicazione Java. Si aprirà una finestra di login in cui si
devono inserire le credenziali dell'istanza di MySQL usata per creare il
database "federgolf", con cui dovrà interagire l'applicazione.
- se le credenziali risultano corrette, si aprirà l'effettiva applicazione
da cui sarà possibile interagire con il database, già popolato con dei dati
iniziali su cui svolgere le operazioni.
